# Dashboard

Esta pasta reúne os artefatos relacionados ao dashboard final do projeto, desenvolvido no Tableau Public a partir da tabela analítica resumida.

O dashboard tem como objetivo sintetizar visualmente os principais resultados analíticos, apoiando a interpretação de padrões de uso, tendências temporais e aspectos operacionais identificados ao longo do pipeline.

A imagem do dashboard final está disponível na pasta [`img/`](https://github.com/chascaldini/mvp-eng-dados-consultas-acervo/blob/main/img/5.6_dashboard_analitico_tableau.png) como evidência visual do resultado final.
