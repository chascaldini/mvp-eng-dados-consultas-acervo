**Dashboard**

Esta pasta reúne os artefatos relacionados ao dashboard final do projeto, construído no Tableau Public a partir da tabela analítica resumida.

O dashboard tem como objetivo sintetizar visualmente os principais resultados analíticos, apoiando a interpretação dos padrões de uso, tendências temporais e aspectos operacionais identificados ao longo do pipeline.

- Prints do dashboard estão disponíveis na pasta `img/` como evidência visual.
- O acesso interativo ao dashboard é feito por meio do link público do Tableau.
