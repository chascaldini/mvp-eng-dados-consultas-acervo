**Dados utilizados**

- **CCSL_2024_Obras_consultadas_por_pesquisadores.csv**  
Dataset público original com registros de consultas a obras do acervo da Fundação Biblioteca Nacional em 2024, conforme disponibilizado no Portal Brasileiro de Dados Abertos.  Fonte oficial: [Portal Brasileiro de Dados Abertos – FBN: Obras consultadas por pesquisadores (2024)](https://dados.gov.br/dados/conjuntos-dados/fbn-obras-consultadas-por-pesquisadores-obras-gerais-manuscritos-e-periodicos)

- **consultas_fbn_tratada.csv**  
  Versão tratada do dataset, gerada após o processo de ETL no Databricks.
